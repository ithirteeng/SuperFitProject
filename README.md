# SuperFitProject

*HITS project*<br/>

Application for people, who enjoy doing sports and being in shape ü•ä<br/>

## Features

- Sensors which track squats, push-ups and passed meters
- Comfortable UI/UX
- Replaceable password buttons for protection from spies üïµÔ∏è
- Ability of uploading photos
- Chart of trains progress
  <br/>

## Technology stack:

- Jetpack Compose
- Coroutines
- MVI
- Clean Architecture
- StateFlow
- Room
- Retrofit2
- Okhttp3
- Encrypted Shared Preferences
- JUnit
- Mockito
- Koin
- Android sensors
